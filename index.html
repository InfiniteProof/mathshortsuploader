TikTok Uploader

TikTok Uploader is a private tool built and operated by a single developer. It publishes pre-made videos to that developer's own TikTok account using TikTok's official Login Kit and Content Posting API. It is not distributed to anyone else, and no other person can use it to post to TikTok.

What it does

The operator authorizes their own TikTok account using TikTok Login Kit (OAuth 2.0). Once authorized, a review screen shows the video that is about to be posted. Before publishing, the operator must manually:

  • Select a privacy level for the post (no option is selected by default)
  • Choose whether comments, duets, and stitches are allowed (all off by default, and disabled entirely if TikTok reports them as turned off in the account's own settings)
  • Confirm consent to publish

Only after all of these are manually set does the Publish button become available. The video is then sent to TikTok using the Content Posting API (Direct Post).

Who uses it

TikTok Uploader is used only by its developer, for their own TikTok account. It is not offered to the public, not sold, and not shared. It connects to exactly one TikTok account: the one its operator personally authorizes through TikTok's login flow.

TikTok integration

  • Login Kit — authorizes the operator's TikTok account (OAuth 2.0)
  • Content Posting API — publishes video content directly to that account (Direct Post)
