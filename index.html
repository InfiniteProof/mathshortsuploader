MathShortsUploader

MathShortsUploader is a content automation platform for creating and publishing animated math explainer videos to social media. It enables authorized users to generate short-form educational videos and publish them directly to their own connected social media accounts.

What it does

The platform uses mathematical animation software (Manim) and AI-assisted scripting to produce high-quality, visually engaging videos covering one math concept per video. Each video is rendered locally and published to the user's authorized accounts via official platform APIs.

Key features

  • Automated video generation from a topic library
  • Synthesized voiceover narration using neural text-to-speech
  • AI-generated titles, descriptions, and tags optimized per platform
  • Direct publishing to TikTok, YouTube, and Instagram via official APIs
  • Review workflow before any content is published
  • Scheduling and topic-deduplication to maintain consistent output

Who it's for

MathShortsUploader is designed for educators, content creators, and developers who want to produce and distribute educational math content at scale. Users connect their own social media accounts and publish only to accounts they personally own and have authorized.

Platform integrations

  • TikTok — Content Posting API (video upload and publishing)
  • YouTube — YouTube Data API v3 (video upload)
  • Instagram — Instagram Graph API (video publish)
